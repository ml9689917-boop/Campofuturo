
# CampoFuturo — versão PRO (moderno & tecnológico)

**Título do site:** CampoFuturo — Previsões de preço e clima  
**Estilo:** campo moderno e tecnológico (verde-esmeralda + dourado)

## O que está neste pacote
- Projeto React + Vite com painel administrativo e dashboard de previsões (preço + clima).
- Integração com Firebase (placeholders). Você needs to create a Firebase project and paste the keys.
- Instruções para deploy no Vercel e criação do administrador.
- Design atualizado: colors, typography and CSS ready for customization.

## Teste local
1. Instale Node.js (>=18) e npm.
2. No terminal dentro da pasta, rode:
   ```bash
   npm install
   npm run dev
   ```
3. Abra `http://localhost:3000`

## .env
Copie `.env.example` para `.env` e cole suas chaves do Firebase.
